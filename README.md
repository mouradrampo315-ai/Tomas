# 🛡️ My First Logic Project (C#)
### Developed by: Badr (Age: 16)

This is my first project focused on learning **C#** logic. I built this program to intelligently validate user data and prevent input errors.

### 🚀 Features:
* **Name Validation:** The program ensures the name contains letters only (no numbers allowed!).
* **Age Validation:** It verifies that the age is a whole number (no fractions/decimals allowed!).
* **Branching Logic:** Displays different sets of questions based on the user's age (Over/Under 18).

### 🛠️ Technologies Used:
* **Language:** C#
* **Concepts:** Input Validation, Logic Branching (If-Else), Type Parsing (TryParse).
